# Utility Functions

This folder contains shared utility functions and helpers used across different algorithms.

## Common Utilities

### Array Utilities

- Array generation
- Array comparison
- Array printing

### Tree Utilities

- Tree node creation
- Tree traversal helpers
- Tree printing/visualization

### Linked List Utilities

- Node creation
- List printing
- List comparison

### Test Utilities

- Assertion helpers
- Performance measurement
- Test case generators

## Usage

Import utilities in your algorithm files:

```typescript
import { arrayEquals, printArray } from '../utils/array-utils';
import { TreeNode, printTree } from '../utils/tree-utils';
```
